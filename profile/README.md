# BlueHero

**AI-native software for improving employee retention.**

We build tools that help companies understand what keeps their people engaged and act on it, with AI as a core part of the product rather than a bolt-on.

## How we build

- **Stack:** TypeScript, React, and Supabase.
- **Delivery:** continuous. We work trunk-based, integrate small changes frequently, and deploy to production daily.
- **Infrastructure:** Vercel for web, Supabase for backend and database, PostHog for feature flags and experimentation.
- **Releases:** we separate deploying code from releasing features, so we can ship continuously and turn functionality on deliberately.
